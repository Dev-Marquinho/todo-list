# To-Do List

Projeto de lista de tarefas desenvolvido com JavaScript puro.

* Funcionalidades

- Adicionar tarefas
- Remover tarefas da tela
- Salvar tarefas com localStorage
- Carregar tarefas automaticamente ao abrir a página

# Tecnologias

- HTML
- CSS
- JavaScript
